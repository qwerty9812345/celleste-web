# 🏠 Celleste Family Site

Семейный сайт с Discord-верификацией. Есть роль **Celleste** на сервере → видишь закрытый раздел (таблица навыков, внутренняя информация).

---

## Пошаговый гайд: получаем 4 значения из Discord

Вам нужно открыть браузер, Discord и скопировать ровно **4 штуки**. Ниже — как найти каждую.

---

### 🔹 Значение №1: APPLICATION ID (он же Client ID)

**Где:** https://discord.com/developers/applications

1. Откройте ссылку выше в браузере
2. Нажмите синюю кнопку **New Application** (справа вверху)
3. Введите название: `Celleste Family`
4. Поставьте галочку "I have read and agree to..." (согласие с условиями)
5. Нажмите **Create**
6. Вы попали на страницу **General Information** вашего приложения
7. **Смотрите на верхнюю карточку.** Там написано "APPLICATION ID" — это длинное число, например `123456789012345678`
8. Нажмите кнопку **Copy** справа от этого числа

✅ Скопировано. Это **Client ID**.

---

### 🔹 Значение №2: Client Secret

**Где:** та же страница **General Information**, чуть ниже

1. Всё на той же странице, где вы только что скопировали APPLICATION ID
2. Чуть ниже есть строка **CLIENT SECRET**
3. Нажмите кнопку **Reset Secret**
4. Появится предупреждение — нажмите **Yes, do it!**
5. Теперь справа появится длинная строка (буквы, цифры, тире)
6. Нажмите **Copy**

⚠️ **Важно:** Client Secret показывается **один раз**. Если закроете страницу — придётся сбрасывать и копировать заново. Скопируйте сразу в блокнот или в скрипт установки.

✅ Скопировано. Это **Client Secret**.

---

### 🔹 Значение №3: Bot Token

**Где:** левое меню → **Bot**

1. Слева найдите раздел **Bot** (иконка в виде робота 🤖)
2. Нажмите **Add Bot**
3. Подтвердите: **Yes, do it!**
4. Вы увидите карточку бота. Сверху — название бота, аватарка.
5. В разделе **Token** нажмите **Reset Token** (если токена нет) или **Copy** (если есть)
6. Подтвердите сброс: **Yes, do it!**
7. Появится строка вида `MTIzNDU2Nzg5MDEyMzQ1Njc4.XyZ_Ab.CdEfGhIjKlMnOp`
8. Нажмите **Copy**

**ВАЖНО: Включите две галочки ниже, иначе ничего не будет работать:**

9. Прокрутите чуть ниже, до раздела **Privileged Gateway Intents**
10. Включите (сделайте серыми/синими):
   - ✅ **SERVER MEMBERS INTENT**
   - ✅ **MESSAGE CONTENT INTENT**
11. Нажмите **Save Changes** (синяя кнопка вверху)

✅ Скопировано. Это **Bot Token**.

---

### 🔹 Значение №4: ID сервера (Guild ID)

**Где:** ваш Discord клиент (программа на компьютере)

**Сначала включите режим разработчика:**

1. Откройте Discord (программа или браузер)
2. Нажмите на шестерёнку ⚙️ внизу слева — **Настройки пользователя**
3. Слева найдите **Расширенные** (или Advanced)
4. Включите **Режим разработчика** (Developer Mode) — переключатель вправо
5. Закройте настройки (крестик)

**Теперь копируем ID сервера:**

6. Посмотрите на **левую панель** Discord — там список серверов (иконки)
7. Нажмите **правой кнопкой мыши** по вашему серверу (тому, где будет семья)
8. В меню выберите **Копировать ID сервера** (самая нижняя опция)
9. ID скопирован в буфер обмена — это число вида `876543210987654321`

**Если у вас ещё нет сервера:**
- Нажмите на зелёный плюсик ➕ внизу списка серверов
- Выберите **Create My Own**
- Назовите сервер (например "Celleste Family")
- Создайте. Теперь кликните по нему правой кнопкой → Копировать ID сервера

✅ Скопировано. Это **Guild ID** (ID сервера).

---

### 🎯 Финальный шаг: запустите скрипт

Теперь у вас в буфере / блокноте 4 значения:

| Что | Пример |
|-----|--------|
| APPLICATION ID | `123456789012345678` |
| Client Secret | `AbCdEfGhIjKlMnOpQrStUvWxYz` |
| Bot Token | `MTIzNDU2Nzg5MDEyMzQ1Njc4.XyZ_Ab.CdEfGhIjKlMnOp` |
| Guild ID | `876543210987654321` |

**Запустите скрипт установки:**

```powershell
# Откройте PowerShell (Win → PowerShell)
cd C:\Users\aleks\Desktop\family-site
.\setup.bat
# или просто дважды кликните на setup.bat в папке
```

Скрипт попросит вас вставить эти 4 значения по очереди — просто нажимайте Ctrl+V или правой кнопкой мыши → Вставить.

## 🔄 После скрипта: ещё 2 действия в браузере

Скрипт сам создаст `.env` и установит зависимости. Но в Discord Developer Portal нужно сделать ещё 2 вещи (займёт 1 минуту):

### 1. Добавить Redirect (куда перенаправлять после входа)

1. Откройте https://discord.com/developers/applications
2. Нажмите на **Celleste Family**
3. Слева выберите **OAuth2**
4. В разделе **Redirects** нажмите **Add Redirect**
5. Вставьте: `http://localhost:3000/auth/discord/callback`
6. Нажмите **Save**

### 2. Пригласить бота на сервер

1. На той же странице **OAuth2** найдите **OAuth2 URL Generator**
2. В **Scopes** отметьте: ✅ `bot` ✅ `identify` ✅ `guilds` ✅ `guilds.members.read`
3. В **Bot Permissions** отметьте: ✅ `Send Messages` ✅ `Manage Roles` ✅ `Read Message History`
4. Внизу появится **Generated URL** — длинная ссылка
5. Нажмите **Copy**, откройте в новой вкладке
6. Выберите ваш сервер → **Continue** → **Authorize** → пройдите капчу

### 3. Создать роль и выдать её

1. В Discord на вашем сервере нажмите ПКМ → **Server Settings** → **Roles**
2. Нажмите **Create Role**
3. Название: `Celleste`
4. Нажмите **Create**
5. Выдайте роль: Server Settings → **Members** → найдите себя → `+` → выберите **Celleste**

## 🚀 Запуск

```powershell
start.bat    # или дважды кликнуть на файл
```

Откройте браузер → **http://localhost:3000**

**Проверка:**
- Нажмите «Войти через Discord»
- Подтвердите вход
- Если у вас есть роль Celleste → появится раздел «Внутреннее» и таблица навыков
- Если роли нет → только публичная часть

## 🤖 Discord бот (опционально)

В отдельном окне PowerShell:
```powershell
npm run bot
```

Команды на сервере: `!role`, `!members`, `!info`, `!help`
